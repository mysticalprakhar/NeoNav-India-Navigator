# NeoNav – India Smart City Navigator

NeoNav is a Data Structures project implemented in C.

It models an Indian transportation network using a weighted graph
and calculates the shortest path using Dijkstra's Algorithm.

## Features

- Graph based city network
- Shortest route calculation
- Modular C structure
- Dataset of Indian cities

## Technologies

- C Programming
- Graph Data Structures
- Dijkstra Algorithm

## Future Enhancements

- Traffic prediction
- Drone delivery routing
- Smart city navigation
